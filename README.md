# ETA-Update-Prediction-Service

## Painpoint
  * MarineTraffic 등의 Reported ETA는 선박이 직접 수정한 ETA만 표시하므로, 기상·항만 혼잡·전쟁 등의 변수 발생 시 실제 지연을 사전에 알기 어렵다.
    =기존 ETA는 실제 지연 가능성을 미리 알려주지 못함.
  1. 글로벌 물류 리스크 심화: 기상 악화 및 지정학적 리스크(전쟁·분쟁 등)로 인한 해상 운송 지연의 상시화
  2. 선제적 대응 및 소명 한계: 지연 발생 시 원인 파악이 늦어 고객사 대응 및 사후 소명 자료 확보에 차질
  3. 높은 솔루션 도입 비용: 외부 변수 예측 인공지능 솔루션은 모두 고가의 유료 툴(기업용 Saas 플랫폼, 연간 사용료 억대로 추정됨)로, 전사적 도입 및 실무 활용에 한계

## 해결방안
  * AIS 데이터, 기상 정보, 항만 혼잡도, 지정학적 리스크 등을 종합 분석하여 AI 기반 예상 ETA 및 지연 가능성을 제공한다.
    = 다양한 외부 변수를 반영한 AI 기반 ETA 예측 제공.
  1. 출발/경유/도착지 기반 경로 자동 추적: 최적경로 예측을 통한 실시간 화물선 예상위치 및 예상운송 경로 시각화
  2. API/AI 기반 외부 리스크 데이터 수집: 경로 내 기상이슈를 API를 통해 수집, 지정학적/정치적 이슈를 공신력 있는 매체 데이터 기반으로 수집
  3. 지연 영향도 시뮬레이션: 수집된 리스크 데이터를 바탕으로 예상 도착일(ETA) 변동성을 예측하고, 지연 원인 기여도를 데이터로 제공

## 기대효과
  1. 물류 관리 비용 혁신: 고가의 외산 솔루션 대체, 사내 인프라와 오픈 데이터를 활용한 비용 제로(Zero)의 선박 추적
  2. 전사적 물류 가시성(Visibility) 확보: 웹 기반 대시보드를 통해 수출 프로젝트 담당자 간 실시간 선적 현황 공유 체계 구축
  3. 대고객 신뢰도 및 만족도 극대화: 도착일 변동에 대한 선제적 모니터링과 데이터 기반 소명으로 지연 리스크 최소화 및 고객 신뢰 확보

## 주요 기능
  1. 프로젝트 관리
    - 해당 프로젝트 내용(선적내용 및 출발/도착정보) 입력/관리
    - 24H 단위 정보/분석 최신화 하여 ETA 변동관리
    - 정보분석 내용 기반 ETA 변동을 계산하므로 신뢰성 향상

## 주요 기술
  1. 정보기반 AI 분석 및 결과물 제작 시 할루시네이션 최소화
    - 엄격한 페르소나 및 규칙 적용
    - 정보 생성자와 검증자 역할 분할하여 할루시네이션 검증
  2. AI 검색 및 생성시간 최적화
    - API 및 AI처리 병렬구조로 정보획득 병렬화
    - 지정학적/정치적 이슈 획득시 주요 매체 크롤링을 병렬처리하여 검색시간 최소화

## 구성 및 방법
  1. 정보 입력
    - 프로젝트명/출발지/경유지(선택)/도착지/출발시ETA/아이템정보 입력
  2. 정보획득 및 분석
    - 출발지/경유지(선택)/도착지 정보를 기반으로 좌표획득
    - searoute 라이브러리 활용하여 최적경로 계산, 10개 좌표로 축약
    - 10개 좌표에 대한 기상정보 획득, 인근국가 특정하여 이슈 획득
    - 획득정보 기반 AI분석, 분석내용/변동ETA/알람 출력
