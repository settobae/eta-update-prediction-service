# 사내 AI 경진대회

## 주제
1. 선박 및 항공기 트래킹 및 자동 문서화 AI Tool 제작
- 사용자 질문에 대한 의도파악, 데이터 획득, 답변 과정에서의 할루시네이션 감소
- 선박 및 항공기 트래킹 정보 획득 및 시각화
- 정해진 문서 양식에 맞게 내용 입력 기능
